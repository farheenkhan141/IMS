Institute Management System
Backend: SpringBoot and PostgreSQL
FrontEnd: Node js (EJS)

Requirements:
IDE (Like Eclipse)
Node js
PostgreSQL



Clone this application in your system:
Enter your username and password of postgres in application.properties file.
1: Create Database "ims"
2: Run backend application
3: Then run this commands in node js:
-> npm i express dotenv ejs axios
-> node server.js
